Disables new Site Health screen from WP Dashboard

== Description ==

The addition of the new Site Health screen can be useful but it exposes a lot of server information that should be kept private. This plugin disables this feature in WordPress. 

Major features in WP Disable Site Health include:

* No setup required!
* Prevents the Site Health screen from exposing critical server information.

== Installation ==

Upload the wp-disable-site-health folder to your wp-content/plugins directory, then Activate it. No configuration needed.
